要求：在满足第三范式的基础上，且不允许主键的一部分被另一部分或其它部分决定。

特定：

1）满足第三范式。

2）所有非主属性对每一个码都是完全函数依赖。

3）所有的主属性对每一个不包含它的码，也是完全函数依赖。

4）没有任何属性完全函数依赖于非码的任何一组属性。

示例：（以下例子 不符合 第BCNF范式）

```
/*学生       老师        课程*/
  Stu      Teacher      Course
  张三       李开复       数据库
```

假设：每个老师只教一门课。

候选码：

```
(Stu,Teacher)-
>
Course

(Stu,Course)-
>
Teacher
```

这两个码由两个属性组成，而且它们是相互交叉的，所以不存在传递依赖，所以为3NF。

但是由于：Teacher-&gt;Course，（主键的一部分被另一部分决定），所以不符合BCNF。
